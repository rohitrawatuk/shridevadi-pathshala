Introduction to CSS

CSS (Cascading Style Sheets) is used to style HTML elements. It defines colors, layouts, fonts, and more.

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Hello, CSS!</h1>
</body>
</html>

CSS Basics

CSS can be added inline, internally, or externally.

/* Inline */
<h1 style="color: red;">Hello</h1>

/* Internal */
<style>
  h1 { color: blue; }
</style>

/* External */
link rel="stylesheet" href="style.css"

Selectors

CSS selectors target HTML elements.

/* Element selector */
p { color: green; }

/* ID selector */
#myDiv { background-color: yellow; }

/* Class selector */
.box { border: 1px solid black; }

/* Group selector */
h1, h2, h3 { font-family: Arial; }

Colors & Fonts

h1 {
  color: #e74c3c;
  font-family: 'Verdana', sans-serif;
  font-size: 24px;
}

p {
  color: rgb(0, 128, 0);
  font-size: 16px;
}

Box Model

Every element is a box with margin, border, padding, and content.

div {
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 2px solid black;
  margin: 20px;
}

Layout

CSS layout techniques include Flexbox and Grid.

/* Flexbox */
.container {
  display: flex;
  justify-content: space-around;
  align-items: center;
}

/* Grid */
.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}

Advanced CSS

Learn animations, transitions, pseudo-classes, and pseudo-elements.

/* Hover effect */
button:hover {
  background-color: #3498db;
  color: white;
}

/* Animation */
@keyframes move {
  0% { transform: translateX(0); }
  100% { transform: translateX(100px); }
}

.box {
  animation: move 2s infinite alternate;
}
